### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `god-clis` 命令了。

- `god-clis init vue-template myVue`
- `god-clis config get`
- `god-clis config set type orgs`
- `god-clis config set registry vuejs-templates`

- `god-clis config set type users`
- `god-clis config set registry YvetteLau`

### 发布

开发完成后，即可发布至 npm.
