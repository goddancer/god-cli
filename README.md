### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `god-clis` 命令了。

- `god-clis init babel-template myNpm`
- `god-clis config get`
- `god-clis config set type orgs`
- `god-clis config set registry babel-template`

- `god-clis config set type users`
- `god-clis config set registry goddancer`

### 发布

开发完成后，即可发布至 npm.
